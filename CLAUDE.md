## Familypro Repo Instructions

Start with [AGENT.md](./AGENT.md) for repository-wide constraints.

For blog-content work, use the repo-local workflow skill at:

- [`.agents/skills/familypro-blog-workflow/SKILL.md`](./.agents/skills/familypro-blog-workflow/SKILL.md)

Apply that workflow when the task involves any of the following:

- Creating a new blog post
- Rewriting, expanding, or refreshing an existing post
- Translating or localizing a post
- Reviewing a post for structure, facts, SEO, or localization quality
- Adjusting blog frontmatter such as `title`, `headline`, `description`, or `summary`

Keep the repo copy as the source of truth. Do not rely on user-level symlinks or machine-local skill installs for this project.
